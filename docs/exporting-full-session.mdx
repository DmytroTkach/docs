**To export the session, do the following:**

1. On the **Session Viewer** page for the selected session, click **Session Forensic Export** under the Player.

2. The Session Forensic Export window opens.

3. Select the **Export full session** option.

4. To include [clipboard text data](#_Vi_ewing_Clipboard) recorded during Client monitoring, select the **Include text data**

5. In the **Export type** drop-down list, select one of the following:

* **Session with audio:** To export monitored sessions with the recorded audio data. Please note that the audio data will be exported only for the Clients with the enabled **Enable audio recording** option.
* **Session without audio:** To export monitored sessions without the recorded audio data.
* **Audio only:** To export only the recorded audio data associated with the filtered sessions. Please note that the audio data will be exported only for the Clients with the enabled **Enable audio recording** option.

**NOTE: Microsoft Media Foundation must be installed on the computer with the Ekran System Server to export only the recorded audio data associated with the sessions. On Windows Server 2008 R2, Desktop Experience must be installed.**

6. Optionally, to protect exported sessions with a password, select the **Protect with password** option and enter the password in the **Password** and **Confirm password** boxes.

7. Click **Export**.

8. The **Forensic Export History** page opens, displaying export progress.

9. As soon as export process finishes, the resulting file becomes available for downloading.

10. Click **Download** to download the file with **Forensic Export** results.
