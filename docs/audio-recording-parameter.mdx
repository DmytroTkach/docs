The Audio Recording parameter allows you to record any audio data from the Client Computer.

**NOTE: The Audio Recording parameter** **is available only if you have** **an activated** [**Enterprise serial key**](https://documentation.ekransystem.com/display/DOC/About+Serial+Keys)**.**

If the **Enable audio recording** option is enabled, any input and output sounds played on the Client computer are monitored and recorded.

The audio data can be monitored and recorded in the following applications:

* Skype (desktop, web, and Microsoft Store application)
* Slack
* Zoom
* Viber
* WhatsApp
* Firefox
* Chrome
* Opera
* Internet Explorer
