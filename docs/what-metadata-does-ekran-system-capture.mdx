**Ekran System Windows Clients** record screen captures and monitor metadata: username, hostname, keystrokes, clipboard text data, application names, active window titles, and activity time.

Optionally, information about URLs visited and USB devices plugged in can also be recorded.

**Ekran System macOS Clients** record screen captures and monitor the following metadata: username, hostname, application names, active window titles, activity time, and URLs visited.

**Ekran System Linux Clients** capture "**exec\***" and "**sudo**" commands  entered, with all their parameters.
