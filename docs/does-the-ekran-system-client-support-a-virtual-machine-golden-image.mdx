Yes, after installation, the unique Client GUID needs to be removed from the Windows Registry (**HKEY\_LOCAL\_MACHINE\SOFTWARE\EkranSystem\Client\AgentGUID**).

After the first connection, each new machine receives a unique ID and is monitored as a separate instance.
