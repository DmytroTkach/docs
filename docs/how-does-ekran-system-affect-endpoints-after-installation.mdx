The average CPU usage by an Ekran System Client is less than 1%.

When there is no activity, the Ekran System Client does not use the CPU at all.

The Ekran System Client uses up to 20 MB of memory. If multiple sessions are monitored, Ekran System allocates 12MB for each new session.
