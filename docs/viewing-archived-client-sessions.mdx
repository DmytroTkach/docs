**To play an archived Client session, do the following:**

1. Log in to the Management Tool as a user with the administrative **Viewing Archived Data** permission.

2. Click the **Monitoring Results** navigation link on the left.

3. On the **Monitoring Results** page, select the **Archived Sessions** tab.

4. On the **Archived Sessions** tab, the list of Client sessions in the archived database is displayed.

5. Click on the required session to open it in the Session Viewer.

6. View the Client sessions in the archived databases in the same way as viewing Client sessions which are not archived.
